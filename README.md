# Shreya Birthday App

A tiny, single-page web app to present a heartfelt birthday message and some fun popups/confetti. Use it to copy or send the message on WhatsApp.

## Features
- Displays the WhatsApp-ready birthday message
- Copy to clipboard
- Open WhatsApp (web or mobile) with the message prefilled
- Birthday popups: modal, toast, and confetti for celebration

## How to use
1. Clone or download the files.
2. Open `index.html` in your browser. (For best experience open on mobile or on desktop with WhatsApp web.)
3. Click **Show Birthday Popups** to see the celebration sequence.
4. Click **Send on WhatsApp** to open WhatsApp with the message prefilled.

## Deploy
- You can host this on GitHub Pages: push the repo to GitHub and enable Pages from the repo settings (serve from `main` branch / root).
- Or serve with any static host (Netlify, Vercel, etc.)

## Customize
- Edit the message inside `index.html` (inside the `<pre>` tag) to change the text.
- Modify popups in `app.js` (using SweetAlert2) to customize styling or sequence.
